# Example Custom Order Presets & Templates

The files in this folder can be imported from the **Custom Order** tab inside the Configuration panel.

## Presets

- `standard_cycle.csgorder.json` – a simple A → AY vowel cycle that loops through every available symbol.
- `bright_swells.csgorder.json` – emphasises brighter vowels (AY, E, I) for energetic leads.

## Templates

- `cycle_36_template.csgtemplate.json` – restores a full 36-semitone setup using the *Standard Cycle* preset with normalization and Slicex markers enabled.
- `random_bounce_template.csgtemplate.json` – loads the *Bright Swells* preset with a seeded random policy and a tighter 24-note range.

### Importing

1. Open **Configuration → Custom Order**.
2. Choose **Import** (for presets) or **Template → Import Template** (for templates).
3. Pick one of these files; they will appear in the dropdown immediately.

Feel free to duplicate these JSON files and edit them to create your own presets and templates!
